# Calculadora ROI PREMIUM

Esta es una calculadora de retorno de inversión para proyectos de Kioskos de Agua y Hielo, Vending y Centros de Llenado.

## Características

- Modelado de hasta 3 escenarios de negocio simultáneos.
- Cálculo de rentabilidad global del proyecto (Suma de modelos).
- Gráficos interactivos de Flujo de Efectivo Acumulado y ROI.
- Generación de reportes en PDF con tablas y gráficas de alta resolución.
- Compatibilidad con modo oscuro y diseño responsive.

## Cómo usar

1. Abre el archivo `index.html` en tu navegador.
2. Configura los modelos de negocio deseados.
3. Observa los resultados en tiempo real.
4. Exporta el reporte final en PDF o copia el resumen para WhatsApp.

## Instalación

Simplemente sube el archivo `index.html` a cualquier servidor web o ejecútalo localmente.
Para publicar en GitHub Pages:
1. Sube este repositorio.
2. Ve a Settings -> Pages.
3. Selecciona la rama `main` y guarda.
